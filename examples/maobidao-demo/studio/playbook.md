# Playbook: 猫笔刀

## Play: 日报型(核心)

- **trigger**:**工作日例行**(周一至周五),当天有大盘行情
- **serves_value**:[信息差, 情感陪伴, 决策辅助]
- **emotional_baseline**:**日常陪伴 + 克制**(无论涨跌都不上头)
- **length**:1200-2000 字
- **material_from**:[当日大盘数据, 当日热点事件, 自己的仓位变化, 即时情绪反应]

### Structure

1. **行情扫读**(开头 100-200 字)
   - 一句话给出今天大盘三个核心指数表现
   - 一两句给当日整体氛围(放量 / 缩量 / 震荡)
   - 不分析原因,先把"今天发生了什么"说清

2. **三到五个具体事件 / 板块短评**(主体 800-1200 字)
   - 每个事件 / 板块用 100-200 字
   - 格式:**事件 → 我个人怎么看 → 仓位行动(若有)**
   - 短评必带个人态度,不是中立播报
   - 可以提及自己的具体仓位变化

3. **当日总判断**(中段 100-200 字)
   - 一句对今天市场情绪的整体诊断
   - 不预测明天,但可以"如果 ___,我打算 ___"

4. **生活感慨收尾**(最后 100 字内)
   - 经常一句无关的生活小事
   - "就这些吧,明天接着看"或类似收

### Opening Patterns
[当天行情数据直入, 一句态度起]

### Closing Patterns
["就这些吧" + 一句日常感慨]

### 注意事项

- **绝不喊涨跌**(永远是"觉得""觉得不""觉得有点")
- **绝不教读者怎么投资**
- 仓位提及要具体(不一定全公开,但提的部分一定真实)
- 段落短,口语化重
- 不要像新闻稿
- "老乡 / 朋友 / 哥们儿"称呼读者出现 1-2 次

## Play: 热点评论型

- **trigger**:大事件发生 / 政策出台 / 行业大新闻
- **serves_value**:[信息差, 决策辅助, 解气]
- **emotional_baseline**:克制 + 个人立场鲜明 + 不上头
- **length**:1500-3000 字
- **material_from**:[事件细节, 历史类比, 亲身经历, 仓位影响]

### Structure

1. **事件还原**(200-300 字)
   - 第一段把事件交代清楚,**不抒情不评论**
2. **个人判断分点**(主体 800-1500 字)
   - 分 3-5 个点逐一展开
   - 每点开头:"我个人觉得 ___" 或 "说实话 ___"
3. **历史类比 / 经验**(300-500 字)
   - 把当前事件类比到历史上类似情况
4. **行动暗示**(200-300 字,不是建议)
   - "我在仓位上是怎么想的"——只说自己,不指导读者
5. **生活感慨收**(100 字内)

### Opening Patterns
[当天行情数据直入, 一句态度起, 时间起]

### Closing Patterns
[一句开放式判断, 一句日常感慨]

### 注意事项

- 不预言市场("一定会 ___")
- 不站队某个利益方
- 历史类比要具体,不能"曾经""以前"

## Play: 主题叙事型

- **trigger**:周末或不定期 / 有想写的生活回忆 / 长篇个人感悟
- **serves_value**:[情感陪伴, 身份认同]
- **emotional_baseline**:温和 + 略带自嘲 + 怀旧但不矫情
- **length**:2500-5000 字
- **material_from**:[个人回忆, 具体场景, 家庭 / 工作 / 早期奋斗经历]

### Structure

1. **场景钩子**:具体场景开篇
2. **故事铺陈**:慢慢讲,大量细节
3. **当时的判断 / 心境**:夹叙夹议
4. **从故事到现在的反思**:不教化,只回到"现在的我怎么看"
5. **回旋收尾**:回到开篇场景或一个具体细节

### Opening Patterns
[小事开(生活), 时间起]

### Closing Patterns
[一句生活感慨]

### 注意事项

- 这种文章里不要塞财经术语
- 自嘲要适度
- 不要"那时候我多么 ___"这种感慨腔——保持具体场景说话
