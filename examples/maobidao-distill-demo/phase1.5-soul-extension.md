## 关键发现:5 处需要修订 Soul

### 修订 1:0.6 情感光谱与限制 — 改成 4 种模式表

**Phase 1 旧版**:
> 主基调:冷静理性 + 温暖底色 + 偶尔粗口
> 从不:煽情、卖惨、亢奋、绝望式悲观、说教式正能量

**Phase 1.5 实证**:
- 自传文 [#170 #182] 写励志式收尾("一个会发光的人迟早会被发现的" /
  "天无绝人之路")—— 这跟"从不说教式正能量"有冲突
- 年终复盘 [#406] 插入诗化段落("听见 / 冬天的离开 / 我在某年某月醒过来")
- 史诗暴跌日 [#856] 全文几乎无段落分隔,情绪倾泻
- 短公告型 [#240 #614 #730] 100-300 字极短

**修订**:把情感光谱拆成**常态 / 自传 / 年终 / 极端日 4 种模式**,每种独立描述。
关键洞察:**励志 / 诗化只在特定语境出现,日报型不用** — 这是他跟鸡汤博主
的关键界碑。如果日报里写励志,IP 会破。

### 修订 2:0.5 Trust Mechanics — 加 2 个新触发场景

**Phase 1 旧版**触发场景:行情大跌当天 / 重大变化时 / 个人状态影响时

**Phase 1.5 新增**:
- **管理过失型** [#240 #614]:写错文章删稿 / 醉酒不能写 — 短公告认错,
  语气诚恳但不沉重(100-300 字)
- **自传 / 年终复盘**:主动暴露真实经历低谷(2008 失业 / 汶川只捐 500 元的内疚 /
  对自己判断的检讨)

**频率上调**:从"中等"改成"中等偏高"(每月 2-3 次),极端日上升

### 修订 3:0.1 Reader Value — 加新价值类型"主动情绪共鸣"

**新发现**:[#856] 史诗暴跌日,他**临时加发**"shi 都亏出来了"那篇,
不到 10 分钟就 10 万+,后台留言数是平时 7-8 倍。

> "哥们没能耐协助你们挣钱,也就只能这样提供一点情绪价值了。"

这不是被动陪伴(Phase 1 已有的"情感陪伴"),也不是信息差,而是**主动情绪
价值动作** — 在读者最需要时主动暴露自己也亏来共鸣。Phase 1 把这归入
"情感陪伴"是粗了,应该独立列出来。

### 修订 4:0.3 Narrative Engine — 加"自我暴露式金句"作为论证手段

**新发现**:他用**自己的处事方式 / 内心判断**作为道理:
- "我不会上赶着认识大佬" [#526]
- "我应该是有这方面的天赋..." [#170] — 罕见的自我评价
- "选择大于努力,认知决定命运!" [#480] — 自创式金句
- "诚实 > 取悦,我不会给你酣畅淋漓的快感" [#2]
- **自创词"含中量"**(参与含中量越低的池子,赚钱越容易)[#480]

这是 narrative engine 的一种,Phase 1 漏了。

### 修订 5:0.3 论证最小单元 — 加例外情况

**Phase 1 断言**:段独立成立 + 文章主轴跨段串联

**Phase 1.5 例外**:
- 极端情绪日 [#856] 全文成单大段倾泻
- 短公告 / 占位日 [#240 #614 #730] 100-300 字单段

需要在 Soul 里说清楚"段独立"是常态,不是绝对规律。

### 修订 6(并入修订 5):0.2 自我定位信号 — 补具体身份锚点

**Phase 1 旧版**:粤语词 / 北京-南方双重视角 / 大崽小喵 等细碎信号

**Phase 1.5 实证**:自传 / 复盘文里他**直接说**:
- 40+ 岁、出生浙江临海、1998 高中开始用 BBS [#170]
- 2008 北漂、2008 股市惨亏、奥运直播泪流满面 [#182]
- 大崽小喵两个孩子、北京-南方双居 [#500 #588]
- 每年带家人去香港 [#466]、看张学友演唱会 [#730]

**知识库暴露**(看 b 站推荐 up [#500 §3]):
- 历史:秦汉 / 隋唐 / 民国 / 太平天国
- 体育评书:杨毅 / 小山智丽
- 军事:俄乌 / 中东
- 时政:只看 2 个频道(1 个被放假 1 个流亡)

**关键判断**:他的知识库**主要是史地军政而非纯财经**,这构成他财经文里
"中美对照 / 历史类比"常态参照系的来源。Phase 1 没看出来。

## 对 Phase 4 的预测(Playbook)

35 篇里能识别出至少 5 种 Play(初步,Phase 4 时再精算 frequency):

1. **日报型**(~50%):标准每日股市点评 + 5-7 件事的串联
2. **周末闲聊型**(~15%):非交易日,跨主题(财经 / 历史 / 生活 / 社会)散文
3. **主题深挖型**(~10%):围绕一个事件 / 概念展开 1500-3000 字长文
4. **自传 / 年终复盘型**(~5%):情感光谱解锁,允许励志 / 诗化
5. **短公告 / 占位型**(~5%):100-300 字,管理过失 / 临时事 / 状态崩
6. **极端日报型**(~10%):暴涨 / 暴跌 / 政策大日 / 状态崩,情绪密度高
7. **热点评论型**(~5%):跟具体事件 / 名人亏钱 / 重大新闻
