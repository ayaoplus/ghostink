## Phase 1 关键发现

### Reader 视角的 6 个问题(distill spec 1.2)

读完 15 篇,作为读者(不是分析者)回答:

**Q1. 为什么有人明天还会来读他?**
因为他**做了塘水过滤**。一个普通人想了解今天的 A 股 / 港股 / 美股 / 政策 / 八卦,
要刷十几个渠道才能拼出全貌,他每晚 22 点把这些浓缩成 5-7 件值得说的事 + 他的判断。
**这是日报型 IP 的核心价值,但他做得比同类好,因为信息密度 + 内幕 + 生活气息的混合。**

**Q2. 我从这里得到什么是新闻网站给不了的?**
- 行业内部细节(基金公司 etf 营销返佣的真实数字、机构传闻)
- 实仓判断(不是分析师空谈,是真有钱有仓位的判断)
- "怎么想这事"的框架(不是结论,是思路)
- 中年男性视角的"避险 > 进取"价值观

**Q3. 作者跟我的关系是什么?感觉像谁?**
**像晚饭后蹲在小区里抽烟的老兄弟**。不是"老师"也不是"大师"。他比我有钱,但写作姿态比我接地气。粤语词、家庭段子、自嘲打了一手好牌——让"高净值 + 行业内人"的形象不脱离日常。

**Q4. 作者相信什么?什么让他愤怒 / 悲伤 / 兴奋?**
- **底层信念**:A 股是"塘主允许的散户互害游戏";中年人要避险;数据 > 情绪
- **愤怒**:体制性的搪塞("一切责任在美方"式)、上市公司的鸡贼(万科百亿回购只买 1 亿)
- **罕见悲伤**:重大不确定性出现时("拿不准还有没有牛周期")
- **兴奋**:几乎不出现。最接近兴奋的是"逮到个有趣事就想说"

**Q5. 作者下结论时,我为什么信(或不信)?**
**信**,因为:
- 他有真金白银的仓位证据
- 他承认错误(不删不洗,主动引述)
- 他承认无知("到时候再说"、"咱也不方便凭空脑补")
- 他的预测有过去的验证记录(读者能查)
- 他用具体数据,不用"印象"

**Q6. 我打开文章前期待什么样的情感体验?**
"今天的塘水我没必要自己游了,他游过了告诉我"——信息确认 + 判断校准 +
意外惊喜(行业八卦)+ 陪伴感(我不孤单)。

### Uniqueness Statement 提炼过程

**第一稿**(基于风格表层):
> "猫笔刀是一个用口语化短句写财经的博主,有自嘲和家庭叙事。"

❌ 太空。任何"日报型口语财经博主"都能套。

**第二稿**(基于身份对照):
> "他是一个 40+ 中年男职业投资者,用'晚饭闲聊'的姿态做财经日报。"

❌ 还是描述,没有差异化锚点。

**第三稿**(基于反差):
> "他做的是'晚饭后蹲在小区跟你抽烟聊塘水'——把一天的市场、政策、八卦
> 加上自家大崽和叔叔的故事全部用短句串起来。可信度来自实仓 + 关系网 +
> 承认无知,不来自任何头衔。"

✓ 有反差锚点(同类博主 vs 他)。

**最终稿**(加一个"反人设"句作为锐度):
增加了 [#2] 里那句"诚实 > 取悦,我不会给你酣畅淋漓的快感"——这是
他跟所有"迎合留言区情绪流财经 V"的本质区别,是不可复制的 IP 信号。

## 对 Phase 2 的预测(distill 多轮迭代的种子)

基于 Soul,我对 Phase 2 的 Form 提取有以下预测——用于 Phase 3
predict-then-compare:

1. **句长**:短句占 ≥60%,中句 ~25%,长句 ~15%(用于分析时)
2. **段长**:大量 2-4 句的极短段;"……" 是关键段间符号
3. **Vocabulary Must Use**:你 / 你们 / 我个人觉得 / 我估计 / 说白了 / 其实 / 哦对了
4. **Vocabulary Forbidden**:老铁 / 家人们 / 各位 / 朋友们 / 加油 / 愿你
5. **Signature Expressions**:就这些了 / 就这些 / 唔知啊 / 咯 / 赢麻了 / 洗洗睡吧 / 鸡贼 / 漏勺 / 真就 / 狗的
6. **Opening Patterns**:"今天行情..." / "周末闲聊,想哪到哪" / 直接砸数据 / 引读者评论
7. **Closing Patterns**:"就这些了 / 就这些吧" / "字数差不多了" / 反讽收尾
8. **讽刺锐度**:主基调 2 级,触发到 3 级是体制性问题或机构鸡贼
9. **emoji 频率**:几乎 0
10. **加粗使用**:中等,用在数据 / 关键判断 / 强调 — 但不滥用

Phase 2 跑完会跟这些预测对比,印证的提升 confidence,违反的标 [REVISE],
新出现的标 [NEW]。
