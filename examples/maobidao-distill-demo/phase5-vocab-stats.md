# Phase 5: Vocabulary Stats(猫笔刀)

> 时间:2026-04-26
> 输入:35 篇 in-context 样本
> 输出:Form 5. Vocabulary Fingerprint 4 层 + 12 条 Signature Expressions

## 词频统计的方法

distill spec 5.1 要求"跨所有文章的高频短语提取"。本次实际操作:

- **没有跑外部 Python 脚本**(distill spec 实现备注:"v1 用 LLM 估算,v2+ 才
  考虑 Python n-gram")
- **基于 35 篇 in-context 样本 LLM 估算**:在每次写 Form 5 时,LLM 在脑里
  扫描 35 篇文章里词的相对出现频率
- **跟 generic AI 中文写作做对照**:挑出"猫笔刀显著高于 generic"的词作
  Must / Often;挑出"generic 高频但他 0 出现"的词作 Forbidden

## v2 升级方向

未来如果要把准度上一个档,可以:
1. **Python n-gram 统计** — 跑全 727 篇,精确到每个词的出现次数
2. **TF-IDF 对比 generic 中文语料** — 严格识别"特征词"
3. **跨作家对比** — 跟王小波 / 韩寒 / 三毛 / 李笑来对比,找"猫笔刀独有"的词

但 v1(distill spec 当前实现)的 LLM 估算已经覆盖了主要的"标志性词" + "禁用词",
精度对实际写作够用。
