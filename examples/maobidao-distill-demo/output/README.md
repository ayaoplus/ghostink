## 关键摘要(读三个文件前的快速预览)

### Soul Uniqueness Statement(导航灯)

> 他不像传统财经博主那样"分析师腔 + 看好/看空二选一",他做的是**晚饭后蹲在
> 小区里跟你抽根烟聊今天塘水**——把当天的市场新闻、政策变化、行业八卦,加上
> 自家大崽和叔叔的故事,全部用短句和省略号串起来,告诉你他怎么看,但不告诉你
> 怎么做。**最反人设**:"诚实 > 取悦,我不会给你酣畅淋漓的快感"

### Form 14 节产出概览

| 节 | 关键产出 |
|---|---|
| 1. Persona Voice | 3 个核心张力:金融术语 × 街头粗口 / 数据严谨 × 自嘲打脸 / 长视野 × 当下生活段子 |
| 2. Reader Cognitive Model | 默认读者智商高 + 已知/未知边界表 + 论证强度档位中 |
| 3. Sentence Rhythm | 短句 ~55% / 中 ~30% / 长 ~15% / 平均 ~22 字 |
| 4. Paragraph Rhythm | 段平均 ~3 句,**"……" 段间符是关键标志(每篇 3-5 次)** |
| 5. Vocabulary Fingerprint | 4 层词表(Must 10 + Often 10 + Never 14 + Forbidden 6 大类) |
| 6. Address Conventions | "我"/"你/你们" + 禁用网红称谓("老铁/家人们") |
| 7. Analogy Style | 赌场+生活+**真人真事**(李佳琦/郭德纲)+ 军事 + 历史 |
| 8. Context Projection | 中美对照常态参照系 + **讽刺锐度 6 级体系**(0/1/2/3/3.5/4 级) |
| 9. Opening Patterns | 7 种,直接砸数据 ~40% 主导 |
| 10. Closing Patterns | 8 种,**"就这些了" ~50% 主导**,励志式仅自传/年终 |
| 11. Emotional Toolkit | 7 情绪表达手法 + 克制系数 ~75% |
| 12. Signature Expressions | 12 条招牌 + **3 自创词**("含中量""塘主""4 倍理论")+ 夸张式串叠 |
| 13. Punctuation & Layout | 加粗中等 + emoji 罕见 + "……"代替 markdown 横线 |
| 14. Prohibition List | 12 条 0 容忍,每条 trace 回 Soul |

### Playbook 7 种 Play(frequency 加起来 ≈100%)

| Play | frequency | 核心场景 |
|---|---|---|
| 日报型 | ~50% | 工作日例行,数据砸脸 + 多新闻串联 + "就这些了" |
| 周末闲聊型 | ~15% | "想哪到哪",跨主题(财经+历史+生活) |
| 极端日报型 | ~12% | 史诗暴跌/暴涨/状态崩,主动情绪共鸣 |
| 主题深挖型 | ~10% | 围绕一个议题打透,可能 1 个大类比贯穿全文 |
| 自传/年终复盘型 | ~5% | 情感光谱解锁,允许励志式收尾 |
| 短公告/占位型 | ~5% | 100-300 字,管理过失 / 临时事 / 状态崩 |
| 热点评论型 | ~3% | 大事件 + 反讽 3 级密度上升 |

加上文末 **Cross-Play Prohibitions**(8 条总禁忌)+ **Self-Check Checklist**
(11 条写完后自检)。
