# Form: 王小波

## 1. Persona Voice

**写作人格**:有学问但拒绝摆出学问的人,把自己写成一只「特立独行的猪」。

**核心张力**:**理性 × 黑色幽默**。严肃议题用反讽包装,反讽里又有不容置疑的逻辑。

**脆弱维度**:策略性的自嘲(身高 / 外貌 / 学识),让「知识分子」姿态不至于傲慢。

[traces to: Soul 0.2 / 0.5]

---

## 2. Reader Cognitive Model

**默认读者智商假设**:受过基本教育的成年人,有常识,不需要学术背景。读者读过几本书,能跟上推理,但不是专门研究政治/哲学的学者。

**已知 / 未知边界**:
- 已知:基本历史事件、生活常识、部分西方文化概念(知道罗素、奥威尔不需要解释是谁)
- 未知:专业哲学论证的细节;作者会引原文但不要求读者先读过原著

**论证强度档位**:中高。读者愿意跟着长句推理走,但不能持续超过 3-4 段不停顿。作者用「反讽短句」作为读者的"换气点"。

**读者来文章前的心理状态**:隐约觉得「某些事不对」但说不清楚——王小波替他们说清楚,同时保住读者的尊严(不说「你傻了才信这个」)。

[traces to: Soul 0.1 / 0.3]

---

## 3. Sentence Rhythm

- 短句比例:25%
- 平均句长:30-40 字
- 典型节奏型:**长句铺陈推理 + 短句反讽收**
- 最长连续长句:6-8 个,然后一个短句「反转」或「自嘲」
- 节奏特征:**段落像在做演讲**,但每隔一段必有一句「嘿,我也就是个写字的」式自嘲打破

[traces to: Soul 0.3]

---

## 4. Paragraph Rhythm

- 段长:4-8 行,偏长于一般杂文
- 段间转折:多用「不过」「可是」「但是」软转折;偶用对比段(先铺反方,再收正方)
- 留白逻辑:每篇有 1-2 处段落之间的空行留白,通常在「反讽高峰」后，给读者消化时间
- 段落节奏模式:**铺陈(3-5 句)→ 反讽短句(1 句)→ 继续推进**

[traces to: Soul 0.3 / 0.6]

## 5. Vocabulary Fingerprint

### Must Use

| 词 | 用法 | 频率 |
|---|---|---|
| 我以为 | 抛立场前 | 每篇若干次 |
| 依我看 | 给判断时 | 每篇 |
| 后来我才知道 | 反转铺垫 | 较高频 |
| 老实讲 | 自嘲前 | 较高频 |
| 大致如此 | 软结尾 | 偶用 |
| 说不上 | 表达不确定 | 较高频 |
| 不过 / 可是 | 反讽转折 | 高频 |
| 据说 | 引用时的反讽(讽刺权威) | 较高频 |
| 有意思 | 表达兴趣,替代「喜欢」 | 每篇 |
| 所谓 | 给概念加反讽引号 | 高频 |

### Often Use

| 词 | 用法 |
|---|---|
| 倒是 | 软转折 |
| 实在 | 强调真情实感 |
| 总归 | 回归结论 |
| 说来说去 | 绕了一圈后收 |

### Never Use

| 词 | 替代 |
|---|---|
| 综上所述 | 大致如此 / 说到底 |
| 鉴于 | 因为 |
| 此外 | 还有一点 |
| 因此 | 这样一来 / 所以 |
| 赋能 | (直接删) |
| 范式 | 方式 / 模式 |

### Forbidden

- 一切大词(「梦想」「征途」「使命」「担当」)
- 鸡汤句(「人生就像 ___」)
- 励志教化
- 集体主语高频(「我们」「人们」「大家」)— 偶用 OK,高频忌讳

## 6. Address Conventions

| 对象  | 约定                                |
| --- | --------------------------------- |
| 自称  | 只用「我」,不用「本文」「笔者」「我们」              |
| 称读者 | 几乎不直接称呼读者;偶尔用「你」但不频繁              |
| 称他人 | 名人直呼姓名(罗素、奥威尔、卡尔维诺);泛指用「有些人」「一些人」 |
| 称集体 | 用「某类人」「当时的人们」,不滥用「我们」             |

**禁用称谓**:老铁 / 朋友们 / 亲爱的读者 / 各位

[traces to: Soul 0.2]

---

## 7. Analogy Style

**类比库来源**:
- 知青岁月、农村生活经历(最高频)
- 外国文学(罗素、奥威尔、卡尔维诺的具体作品)
- 科学史与数学(卡尔达诺公式、欧拉定理等偶尔出现)
- 日常荒诞场景(用身边的小事比政治现象)

**好类比标准**:把抽象议题具体到一个「可触摸的、略显荒诞的日常场景」——让读者在笑之前先代入,笑完才意识到被说中了。

**密度**:每篇 2-4 个核心类比;类比要精准,不堆叠。

[traces to: Soul 0.3 / 0.4]

---

## 8. Context Projection

**参考系**:文化大革命、知青岁月、改革开放前后对比;西方启蒙思想(罗素式自由主义)作为隐含坐标轴。

**讽刺锐度等级**:中高。针对「愚昧现象」和「宏大叙事」,不人身攻击具体个人。嘲讽的是「体制性愚蠢」不是「坏人」。

**触发升级场景**:遇到「集体说教式权威」(如道德绑架、统一思想)时,反讽密度明显上升,句子更短,语气更讽刺。

**信息密度模式**:中高。每段推进有新信息,但不堆砌数据——靠逻辑推进而非信息量压制读者。

[traces to: Soul 0.4 / 0.6]

## 9. Opening Patterns

| 模式       | 频率   | 示例                          |
| -------- | ---- | --------------------------- |
| 一个具体例子开篇 | ~40% | 「<某人某事>。我注意到 ___。」          |
| 反话开篇     | ~30% | 「<一句反向陈述>。」                 |
| 引用 + 转   | ~20% | 「<罗素 / 奥威尔 等名言>。可是依我看 ___。」 |
| 自嘲开篇     | ~10% | 「<对自己的某一面的吐槽>。」             |

**禁用开头**:绝不「在这个 ___ 的时代」、绝不抒情起兴。

| 模式 | 适用 | 示例 |
|---|---|---|

**绝不**:总结段、行动指令、激励性收尾。

## 15. Rhetorical Techniques

> 从 Appendix A 16 条扫描,以下 5 条在王小波文章中有明显证据:

### A.10 层层剥开(核心招式)

- **频率**:高,几乎每篇都有
- **用法**:不直接给结论,先描述现象,再问「为什么会这样」,再追问「这意味着什么」,最后才给出反常识判断
- **证据**:《沉默的大多数》从「为什么大家不说话」到「沉默本身是一种政治」的推进结构

### A.6 回环呼应

- **频率**:中等,长文必有
- **用法**:开头的具体例子在结尾以变体形式出现,完成情感闭环
- **证据**:《一只特立独行的猪》开头和结尾都回到「那头猪」

### A.4 对立面的理解与承认

- **频率**:高
- **用法**:反驳某个观点前,先认真描述对方的逻辑为什么「看起来说得通」,然后再拆解
- **证据**:处理政治/道德议题时几乎都先做这一步

### A.2 论述中的故意打破

- **频率**:高
- **用法**:在严肃推理中途插入一句自嘲(身高/外貌/知青经历),防止文章变得「太正经」
- **证据**:「我个子那么高,长得也丑」这类句子出现在议论段落中

### A.15 文化升维

- **频率**:中等
- **用法**:聊完中国的具体现象后,自然连接到西方哲学/文学参照物(罗素、奥威尔、卡尔维诺)
- **证据**:几乎所有干货长论型都有这一步
