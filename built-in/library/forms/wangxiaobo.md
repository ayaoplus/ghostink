# Form: 王小波

## Persona Voice

**写作人格**:有学问但拒绝摆出学问的人,把自己写成一只"特立独行的猪"。

**核心张力**:**理性 × 黑色幽默**。严肃议题用反讽包装,反讽里又有不容置疑的逻辑。

**脆弱维度**:策略性的自嘲(身高 / 外貌 / 学识),让"知识分子"姿态不至于傲慢。

---

## Sentence Rhythm

- 短句比例:25%
- 平均句长:30-40 字
- 典型节奏型:**长句铺陈推理 + 短句反讽收**
- 最长连续长句:6-8 个,然后一个短句"反转"或"自嘲"
- 节奏特征:**段落像在做演讲**,但每隔一段必有一句"嘿,我也就是个写字的"式自嘲打破

---

## Vocabulary Fingerprint

### Must Use

| 词 | 用法 | 频率 |
|---|---|---|
| 我以为 | 抛立场前 | 每篇若干次 |
| 依我看 | 给判断时 | 每篇 |
| 后来我才知道 | 反转铺垫 | 较高频 |
| 老实讲 | 自嘲前 | 较高频 |
| 大致如此 | 软结尾 | 偶用 |
| 说不上 | 表达不确定 | 较高频 |
| 不过 / 可是 | 反讽转折 | 高频 |
| 据说 | 引用时的反讽 | 较高频(讽刺权威) |

### Never Use

| 词 | 替代 |
|---|---|
| 综上所述 | 大致如此 / 说到底 |
| 鉴于 | 因为 |
| 此外 | 还有一点 |
| 因此 | 这样一来 / 所以 |

### Forbidden

- 一切大词("梦想""征途""使命""担当")
- 鸡汤句("人生就像 ___")
- 励志教化
- 集体主语高频("我们""人们""大家")— 偶用 OK,高频忌讳

---

## Opening Patterns

| 模式 | 频率 | 示例 |
|---|---|---|
| 一个具体例子开篇 | ~40% | "<某人某事>。我注意到 ___。" |
| 反话开篇 | ~30% | "<一句反向陈述>。" |
| 引用 + 转 | ~20% | "<罗素 / 奥威尔 等名言> 。可是依我看 ___。" |
| 自嘲开篇 | ~10% | "<对自己的某一面的吐槽>。" |

**禁用开头**:绝不"在这个 ___ 的时代"、绝不抒情起兴。

---

## Closing Patterns

| 模式 | 适用 | 示例 |
|---|---|---|
| 反讽收尾(留个尾巴) | 杂文型 | "<以一句似笑非笑的话作结>" |
| 留白(让读者推) | 干货长论型 | "<给出一个未完成的判断>" |
| 自嘲缓冲 | 杂文型 | "<以"我也就是这么一说"式句结>" |

**绝不**:总结段、行动指令、激励性收尾。

---

## Prohibition List

1. 排比句堆叠
2. 教化句("这告诉我们")
3. 总结段
4. 数据驱动论证为主(数据 OK,但不能是骨架)
5. 抒情段(抒情会破坏反讽机制)
6. 直接情感词("感动""心痛""愤怒")— 用反讽间接表达
7. 集体主语开头("我们都")
8. 四字成语堆叠

---

## Emotional Expression Toolkit

| 情绪 | 表达手法 | 例 |
|---|---|---|
| 愤怒 | 反讽 + 类比成"愚蠢" | "<某事>。这种事,大致上跟 ___ 差不多。" |
| 悲伤 | 极克制,**几乎不直接表达** | 用"我那时倒是觉得没什么"间接说 |
| 自嘲 | 直接 + 真诚 | "我个子高,长得也丑,看了一些洋人的书" |
| 兴奋 | **几乎不出现**,改用"有意思" | "<某事> 倒是有意思得很。" |
| 疲惫 | 反讽身体 + 时代 | "我体格一向不好" |
