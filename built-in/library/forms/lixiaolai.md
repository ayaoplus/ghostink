# Form: 李笑来

## Persona Voice

**写作人格**:在你前面探路并喊你"快点跟上"的引路人。**自信但反复强调"我也错过"**。

**核心张力**:**布道感 × 平易近人**。要立威要建权,但用通俗例子和大量"你看"消解距离。

**脆弱维度**:策略性。在关键转折处放出过往错误,软化"权威"姿态。

---

## Sentence Rhythm

- 短句比例:55%
- 平均句长:15-22 字
- 典型节奏型:**陈述长句 + 强调短句 + 例子展开**
- **强调句单句成段**(关键特征):重要观点会用一行单独成段,加粗
- 节奏特征:**用空行制造停顿**,让重要句"被看见"

## Vocabulary Fingerprint

### Must Use

| 词 | 用法 | 频率 |
|---|---|---|
| 重新定义 | 引入新概念时 | 几乎每篇 |
| 你看 | 转向例子或强调 | 高频 |
| 注意 | 强调标记 | 高频 |
| 所谓 | 解构既有概念 | 每篇若干 |
| 事实上 | 给反常识判断前 | 高频 |
| 即便如此 | 让步式转折 | 较高频 |
| 再换个角度 | 切换视角 | 较高频 |
| 我们 | 集体主语,创造共识感 | 高频 |
| 一定 | 给确定性判断 | 较高频 |
| 必须 | 给行动指令 | 较高频 |
| 其实 | 拆穿表象 | 较高频 |

### Never Use

| 词 | 替代 |
|---|---|
| 大概 | 几乎一定 |
| 也许 | 大致 |
| 似乎 | (避免软化) |
| 反正 | (太随意,避免) |

### Forbidden

- 反讽与讥讽语气(他要立威要建权,反讽损权威)
- 抒情段
- 闲聊段
- 自我贬低(自嘲 OK,但不能"我也就那样")

## Opening Patterns

| 模式 | 频率 | 示例 |
|---|---|---|
| 反常识断言开 | ~35% | "<大多数人以为 X,实际上是 Y>。" |
| 重定义开 | ~30% | "什么叫 ___?让我们先重新定义这个词。" |
| 抛问题开 | ~20% | "<一个直击痛点的问题>?" |
| 直接观点开 | ~15% | "<一句立场陈述>。" |

**禁用开头**:抒情、反讽、闲聊式("最近 ___")。

---

## Closing Patterns

| 模式 | 适用 | 示例 |
|---|---|---|
| 回到核心概念 + 行动指令 | 教程型 | "<重申概念>。从今天起,___。" |
| 总结 + 下一步 | 干货长论型 | "<浓缩核心>。下一篇我们继续 ___。" |
| 反复强调单句 | 教程型 | "**记住:___。**"(单句成段) |

**绝不**:开放式留白、反讽收尾、抒情结尾。

---

## Prohibition List

1. 反讽 / 讥讽
2. 抒情
3. 闲聊式段落
4. 模糊判断("大概""也许"高频)
5. 长句堆叠超过 3 句不停
6. 段落超过 4 行(他偏好短段)
7. 不用强调标记的关键观点(关键句必加粗或单句成段)

| 情绪 | 表达手法 | 例 |
|---|---|---|
| 自信 | 直接断言 + 加粗 | "**这就是关键**" |
| 严肃 | 单句成段 + 句号收尾 | "注意。" |

## 视觉强调约定

李笑来 form 严重依赖**视觉节奏**:

- 关键观点 → `**加粗**`(每篇 5-10 处)
- 重要单句 → 独占一段,前后空行
- 概念定义 → 加粗 + 独占一段
- 列表用得多(行动 / 维度 / 步骤)

写作时不能省略这些标记——它们是 form 的一部分,不是装饰。
