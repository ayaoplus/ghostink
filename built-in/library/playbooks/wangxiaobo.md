# Playbook: 王小波

## Play: 杂文型

- **trigger**:对某事 / 某种现象 / 某个流行观点想表态
- **serves_value**:[思想刺激, 反愚昧的快感, 智性娱乐]
- **emotional_baseline**:嘲讽 + 自嘲 + 冷静
- **length**:1500-3500 字
- **material_from**:[亲历的具体事, 引用的书或人, 反常识的观察]

### Structure

1. **抛观点**:开头一句反常识或反讽断言("我以为 ___")
2. **支撑展开**:讲一个具体故事或例子
3. **反例 / 对手意见**:认真处理反对意见,而非嘲笑
4. **推进**:从反例往前推,论证更深
5. **自嘲缓冲**:在论证最重时,用一句自嘲消解学究气
6. **留白收尾**:不给结论,给一句似笑非笑的话

### Opening Patterns
[反话开篇, 引用 + 转, 一个具体例子开篇]

### Closing Patterns
[反讽收尾, 自嘲缓冲, 留白]

### 注意事项

- 反讽要有具体落点,不能只是空话
- 自嘲不能频繁(每段都自嘲就成了讨好)
- 如果论点真的严肃,不要怕被骂"装",但要用例子支撑

## Play: 干货长论型

- **trigger**:对一个议题想长篇展开 / 有几个观点要串起来
- **serves_value**:[思想刺激, 框架认知]
- **emotional_baseline**:严肃 + 反讽缓冲
- **length**:3500-6000 字
- **material_from**:[多个例子, 引用的书, 个人经历, 历史事件]

### Structure

1. **总钩子**:从一个具体场景或反常识陈述开篇
2. **议题边界**:界定要谈什么 / 不谈什么(明确范围)
3. **3-5 个分论点**:每个分论点配 1-2 个例子,反讽点缀
4. **每段之间用反讽承上启下**:不要"首先 / 其次"那种公文连接
5. **承认局限**:在合适位置承认"我不能解决所有问题"
6. **留白收尾**:开放式,让读者推

### Opening Patterns
[反话开篇, 一个具体例子开篇, 引用 + 转]

### Closing Patterns
[留白, 反讽收尾]

### 注意事项

- 分论点之间要有内在逻辑,但不要用公文体连接
- 反讽是节奏,不是内容——每隔几段必须有一处
- 严肃论证时要敢于严肃,但不要变成教训人
