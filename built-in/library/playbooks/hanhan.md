# Playbook: 韩寒(博客时代)

## Play: 热点评论型

- **trigger**:发生了一个新闻 / 公共事件 / 流行说法
- **serves_value**:[解气, 反权威认同, 娱乐]
- **emotional_baseline**:辛辣、反讽、克制不上头
- **length**:800-2000 字(短为主)
- **material_from**:[新闻事件, 一个具体细节, 类比联想]

### Structure

1. **事件直入**:第一句陈述事件或一句反讽起手("<某事>。")
2. **态度短句**:第二段 / 第三段抛态度,几个短句堆叠
3. **荒诞类比**:用一个看似离题的荒唐类比,把事件本质暴露
4. **自嘲缓冲**:中间穿插一两句"我也没什么文化"式自嘲
5. **反讽收尾**:最后一段以一个反讽收,不教化不总结

### Opening Patterns
[新闻或事件直入, 反问开]

### Closing Patterns
[反讽收(荒诞类比), 短陈述收]

### 注意事项

- 段落短,空行多
- 句号比逗号多
- 绝不堆四字成语
- 类比要荒诞但精准——荒唐到笑,但又确实戳中本质
- 自嘲不能频繁,每篇 1-2 处足够

## Play: 杂文型

- **trigger**:有个不针对具体新闻的小观察 / 牢骚 / 即兴想法
- **serves_value**:[娱乐, 反权威认同, 身份认同]
- **emotional_baseline**:玩世不恭,内里清醒
- **length**:1000-2500 字
- **material_from**:[日常观察, 私人经历, 自嘲题材]

### Structure

1. **跳跃式开**:从一个看似无关的小事或自嘲开始
2. **联想 / 闲聊感**:中间几段像在闲扯,但每段都埋一个点
3. **荒诞反讽点缀**:每隔一段必有一处反讽
4. **回归 / 留白**:绕回最初,或者突然一句态度结束

### Opening Patterns
[反讽自嘲开, 短陈述句开]

### Closing Patterns
[短陈述收, 自嘲收]

### 注意事项

- 跳跃要快,但每跳都有线
- 闲扯感是表面,内核要扎实
- 不要在杂文里出现"今天我要谈 ___"这种公文式开篇
