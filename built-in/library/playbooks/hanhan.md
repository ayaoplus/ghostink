# Playbook: 韩寒(博客时代)

## Play: 热点评论型

- **frequency**:~60%(博客时代主力打法)
- **trigger**:发生了一个新闻 / 公共事件 / 流行说法,当天或次日有感而发
- **serves_value**:[解气, 反权威认同, 娱乐]
- **emotional_baseline**:辛辣、反讽、克制不上头
- **length**:800-2000 字(短为主,博客读者等不了长文)
- **material_from**:[新闻事件, 一个具体细节, 类比联想]
- **publish_cadence**:追热点时效,通常事件发生 24-48 小时内发

### Structure(段落级流程)

1. **事件直入** — 第一句陈述事件或一句反讽起手(「<某事>。」);不铺垫,不交代背景,读者已知热点
2. **态度短句** — 第二 / 三段抛态度,3-4 个短句堆叠;每句一个逗号或句号,不用长句
3. **荒诞类比** — 用一个看似离题的荒唐类比,把事件本质暴露;这是全篇最重要的段落
4. **自嘲缓冲** — 中间穿插 1-2 句「我也没什么文化」式自嘲,消解「批评者」的包袱
5. **反讽收尾** — 最后一段以一个反讽收,不教化不总结;可以是一个反问,也可以是一句冷淡的陈述

### Core Characteristics(灵魂识别码)

1. 段落极短(1-3 行),空行极多——「博客感」的视觉标志
2. 荒诞类比必须「荒唐到可笑,但又确实戳中本质」——笑完才觉得「这比喻准」
3. 绝不堆四字成语(这是韩寒 form 的硬红线)
4. 自嘲每篇 1-2 处,超过就变成讨好
5. 结尾不总结、不教化——「点到即止」是全篇节奏的延续

### Opening Patterns
[新闻事件直入, 反问开]

### Closing Patterns
[反讽收(荒诞类比), 短陈述收]

### Anti-Patterns

- 开头交代背景超过 2 句(读者已知热点,不需要铺垫)
- 荒诞类比过于「合理」(失去荒诞的冲击感)
- 自嘲超过 2 处(变成讨好)
- 结尾出现教化或行动指令
- 段落超过 4-5 行(破坏博客节奏感)

[traces to: Soul 0.4 / 0.6]

## Play: 杂文型

- **frequency**:~40%
- **trigger**:有个不针对具体新闻的小观察 / 牢骚 / 即兴想法
- **serves_value**:[娱乐, 反权威认同, 身份认同]
- **emotional_baseline**:玩世不恭,内里清醒
- **length**:1000-2500 字
- **material_from**:[日常观察, 私人经历(赛车/写作圈), 自嘲题材]
- **publish_cadence**:不规律,有想法就写,不强求时效

### Structure(段落级流程)

1. **跳跃式开** — 从一个看似无关的小事或自嘲开始;不解释「我今天要谈什么」
2. **联想 / 闲聊感** — 中间几段像在闲扯,但每段都埋一个点;靠句末语气词(嘛 / 啊)制造「聊天」感
3. **荒诞反讽点缀** — 每隔 2-3 段必有一处反讽;反讽是节奏,不是主题
4. **回归 / 留白** — 绕回最初的话题,或突然一句态度结束;不给结论

### Core Characteristics(灵魂识别码)

1. 跳跃感是刻意的——跳跃快,但每跳都有内在线索(读者感觉「飘」但不散)
2. 闲扯感是表面,内核扎实——表面是「随便聊聊」,内里有一个核心态度
3. 私人经历(赛车 / 出书 / 被骂)是最常用的素材来源
4. 绝不「今天我要谈 ___」这种公文式开篇
5. 结尾通常比开头「重一点」——绕了一圈,最后一句有分量

### Opening Patterns
[反讽自嘲开, 短陈述句开]

### Closing Patterns
[短陈述收, 自嘲收]

### Anti-Patterns

- 开篇明确说「今天我要谈 ___」(破坏跳跃感)
- 中间段落出现「首先其次」这类连接词
- 跳跃超过 4-5 段没有任何反讽/线索回收(真的散了)
- 结尾给励志句或行动指令

[traces to: Soul 0.3 / 0.6]

---

## Cross-Play Prohibitions(总禁忌 — 所有 Play 共用)

1. **不堆四字成语**（连续 2 个即违规）— 因为违反 Soul 0.4(不信宏大叙事)
2. **不写励志 / 感悟段落** — 因为违反 Soul 0.6(绝不写「我终于明白了」类)
3. **不向某个立场靠拢**（不被任何阵营收编）— 因为违反 Soul 0.5(与各方保持距离)
4. **不直接煽情**（不是反讽包装的情感不出现）— 因为违反 Soul 0.6(愤怒和煽情都不出现)
5. **不写总结段** — 因为违反 Soul 0.6(情感契约是「点到即止」)
6. **不假装比读者高明**（自嘲是必需的）— 因为违反 Soul 0.2(绝不摆「过来人」姿态)

---

## Self-Check Checklist(写完任意一篇后的自检)

- [ ] 是否有荒诞类比支撑核心态度(不是空骂)?
- [ ] 段落是否够短(主要是 1-3 行,没有超过 5 行的长段)?
- [ ] 是否有自嘲(1-2 处,不超过)?
- [ ] 结尾是否无总结 / 无教化 / 无励志?
- [ ] 是否避免了四字成语堆叠?
- [ ] 是否避免了 Cross-Play Prohibitions 全部 6 条?
- [ ] 语气是否是「同龄人聊天」而不是「批评家评论」?
- [ ] 整体读感是否像「敢说但不假装高明的同龄人说话」?
