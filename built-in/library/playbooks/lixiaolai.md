# Playbook: 李笑来

## Play: 教程型(核心)

- **trigger**:教读者一个新方法 / 概念 / 思维框架
- **serves_value**:[框架认知, 决策支持, 教程性内容]
- **emotional_baseline**:严肃 + 自信 + 引导感
- **length**:2000-4000 字
- **material_from**:[概念定义, 自己的实操经验, 具体案例]

### Structure

1. **抛反常识断言**(开头 100-200 字)
   - "大多数人以为 X,实际上是 Y"
   - 或重新定义关键词:"什么叫 ___?"

2. **重定义关键概念**(200-400 字)
   - 给出新的定义,通常加粗或单句成段
   - **"重新定义 ___"**(单句成段加粗)

3. **从概念推方法论**(主体 1000-1500 字)
   - 分点展开,通常 3-5 个维度
   - 每点:陈述 → "你看 ___" 转例子 → 强调短句收
   - 关键句加粗或单句成段

4. **案例佐证**(500-800 字)
   - 自己的实操数据 / 经历
   - 名人案例(罗素、查理芒格、纳西姆塔勒布等)

5. **行动指令收尾**(200-300 字)
   - "从今天起,你必须 ___"
   - 通常列点,3-5 个具体行动
   - 最后一句单句成段加粗:**"<重申核心概念>。"**

### Opening Patterns
[反常识断言开, 重定义开]

### Closing Patterns
[回到核心概念 + 行动指令, 反复强调单句]

### 注意事项

- **关键观点必加粗或单句成段**(不是装饰,是 form 的一部分)
- 段落短(不超过 4 行)
- 大量 "你看" "注意" "事实上"
- 行动指令必须具体可执行
- 不要反讽不要抒情
- 不要承认"也许""可能"过多——他要立威

## Play: 干货长论型

- **trigger**:一个核心概念值得展开成长篇 / 多个维度需要串起来
- **serves_value**:[框架认知, 身份认同]
- **emotional_baseline**:严肃 + 庄重 + 反复强调
- **length**:3500-6000 字
- **material_from**:[多个相关概念, 自己长期观察, 大量引用]

### Structure

1. **总钩子**:反常识断言 + 解释为什么这事重要
2. **概念重定义**:核心词从 1 到 N 重新理解
3. **从概念推到 5-7 个维度**:每个维度展开
4. **每个维度都有"你看"做例子转折**
5. **关键观点反复强调**(同一观点用不同方式说 2-3 次)
6. **最后回到总概念,行动指令收尾**

### Opening Patterns
[反常识断言开, 重定义开]

### Closing Patterns
[总结 + 下一步, 回到核心概念 + 行动指令]

### 注意事项

- 重复是这种文体的特征——同一观点用不同角度反复说
- 不要怕长,但每段必须短
- 大量列表和加粗
- 不能让读者觉得"在讲故事"——必须始终是"在传递认知"

## Play: 杂文型

- **trigger**:对某事的随感 / 不需要完整教程结构 / 短篇评论
- **serves_value**:[框架认知, 身份认同]
- **emotional_baseline**:略放松,但仍带"重定义"色彩
- **length**:1500-2500 字
- **material_from**:[即兴感悟, 一个引发思考的小事, 一句别人的话触发]

### Structure

1. **触发事件 / 一句话起**:某事 / 某句话
2. **从事件提炼核心问题**
3. **用 2-3 个角度展开**:每个角度短
4. **回归概念**:必须把杂感落到一个可命名的概念上
5. **强调短句收**

### Opening Patterns
[抛问题开, 直接观点开]

### Closing Patterns
[反复强调单句, 回到核心概念]

### 注意事项

- 即使是杂文,也不能纯随性——必须有"重定义"动作
- 不能闲聊式
- 关键短句仍然加粗或单句成段
