# Playbook: 三毛

## Play: 主题叙事型

- **frequency**:~70%(她的主力打法,撒哈拉系列几乎全是这种)
- **trigger**:有一段完整的经历想写出来 / 怀念某个人或地方 / 一个事件触发了情感 / 想记录某个已经过去的人生阶段
- **serves_value**:[情感陪伴, 信息差, 身份认同]
- **emotional_baseline**:温柔、缓慢、有重量
- **length**:2500-5000 字(篇幅充裕,允许铺陈)
- **material_from**:[亲历的具体场景, 当时的对话, 物件细节, 情绪记忆]
- **publish_cadence**:不规律;一段经历沉淀够了才写

### Structure(段落级流程)

1. **场景钩子** — 用一个具体场景或一句感慨作为开篇,把读者带进时空(「那年我和荷西在 ___」);第一句必须是感官可触碰的具体细节
2. **铺陈与展开** — 慢慢讲事情的经过,大量细节(环境描写、对话、动作并行);不要急着抛核心情感,细节本身是情感的载体
3. **情感聚焦** — 在某一个具体物件 / 对话 / 动作上停下来,让情绪「聚焦」;还不是最重的一句
4. **情感着陆** — 用短句「着陆」最重的情绪(「我那时候哭了一会儿。」);这是全篇最短的句子,也是最重的
5. **回旋或留白** — 不强制总结,要么回到开篇的场景做呼应(回环 A.6),要么以一个无答案的问句 / 一个景物结尾

### Core Characteristics(灵魂识别码)

1. 物件 / 食物 / 天气 / 颜色是叙事骨架——情感通过具体物件传递,不直接喊
2. 情感着陆句必须极短(≤10 字)——长铺垫 + 短着陆是这个 play 的核心节奏
3. 绝不用上帝视角分析自己或他人——只叙述当时发生了什么
4. 对话是场景的一部分,不是「证据」——不用对话「证明」某个论点
5. 开头和结尾之间有情感弧度——不是信息量增加,是情绪质地从「引入」到「沉下去」

### Anti-Patterns

- 开篇用抽象感慨(「孤独是一种 ___」)而不是具体场景
- 中段出现「这件事告诉我 ___」这类总结句
- 情感着陆句太长(超过 20 字就会失去着陆感)
- 回旋/呼应做得太刻意(读起来像「现在我来解释开头的那个意象」)
- 对他人做刻薄批评或上帝视角分析

## Play: 杂文型

- **frequency**:~30%
- **trigger**:有一个想法 / 感慨,但不需要完整故事支撑 / 对某种生活状态有感而发
- **serves_value**:[情感陪伴, 身份认同]
- **emotional_baseline**:轻盈,带一点哲理感但不教化
- **length**:1000-2500 字
- **material_from**:[即兴感悟, 日常观察, 一个引发思考的小事, 别人对她说的一句话]
- **publish_cadence**:不规律,有感而发,比主题叙事型产出更频繁

### Structure(段落级流程)

1. **触发** — 从一件具体小事或一句话出发(可以是别人对她说的话,可以是路上看到的)
2. **联想** — 跳跃式联想,带读者到几个相关的场景或回忆;联想之间不用显式连接词
3. **回归** — 绕回最初的小事,但视角已经变了——不是「所以这件事说明 ___」,是「我现在看这件事,发现 ___」
4. **留白** — 不给「答案」,给一个画面或感慨收尾;可以是一个温柔的反问

### Core Characteristics(灵魂识别码)

1. 跳跃有内在线索——不是真的散漫,只是不显式连接;读者感觉「漂」但不失
2. 情绪连续性在跳跃中保持——即使场景变了,「心理温度」不变
3. 比主题叙事型短,但情感密度可以更高
4. 不给答案——只命名感受,让读者自己对号入座
5. 最后一句通常比整篇轻——不是「最重要的一句」,而是「把手放开」的感觉

### Anti-Patterns

- 跳跃之间用「首先其次」或「另外」这类显式连接(破坏「漂流感」)
- 中途出现分析性段落(「这说明 ___」)
- 收尾太重——最后一句变成「金句」或「人生道理」
- 跳跃超过 4 段没有任何回归/线索(真的散了)

## Cross-Play Prohibitions(总禁忌 — 所有 Play 共用)

1. **不教育读者怎么活** — 因为违反 Soul 0.2(从不教育读者,从不给「人生道理」)
2. **不写励志感悟段落** — 因为违反 Soul 0.6(绝不写励志感悟段落)
3. **不假装坚强**（哭就是哭,不掩饰）— 因为违反 Soul 0.5(常态化的真实脆弱)
4. **不空洞赞美流浪**（流浪不浪漫,只是她自己的选择）— 因为违反 Soul 0.5(会瞬间破坏信任)
5. **不对他人刻薄**（悲悯代替批评）— 因为违反 Soul 0.4(真情高于体面)
6. **不用数据分析腔**（「根据 ___」「研究表明 ___」）— 因为违反 Soul 0.3(亲身经历主导)

## Self-Check Checklist(写完任意一篇后的自检)

- [ ] 是否有具体的感官细节(物件 / 颜色 / 声音 / 食物)?
- [ ] 情感着陆句是否足够短(≤10-15 字)?
- [ ] 是否避免了「这件事告诉我 ___」这类分析性总结?
- [ ] 结尾是否开放(不是结论,是画面或感慨)?
- [ ] 是否没有对任何人做刻薄批评?
- [ ] 「我」的感受是否在场(没有回避,也没有喊)?
- [ ] 整体读感是否像「一个朋友坐在你旁边轻声说话」?
- [ ] 是否避免了 Cross-Play Prohibitions 全部 6 条?
