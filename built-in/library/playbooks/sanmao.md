# Playbook: 三毛

## Play: 主题叙事型

- **trigger**:有一段完整的经历想写出来 / 怀念某个人或地方 / 一个事件触发了情感
- **serves_value**:[情感陪伴, 信息差, 身份认同]
- **emotional_baseline**:温柔、缓慢、有重量
- **length**:2500-5000 字(篇幅充裕,允许铺陈)
- **material_from**:[亲历的具体场景, 当时的对话, 物件细节, 情绪记忆]

### Structure

1. **场景钩子**:用一个具体场景或一句感慨作为开篇,把读者带进时空("那年我和荷西在 ___")
2. **铺陈与展开**:慢慢讲事情的经过,大量细节,环境描写、对话、动作并行;不要急着抛核心情感
3. **情感着陆**:在某一个具体物件或动作上停下来,用短句"着陆"情绪("我那时候哭了一会儿。")
4. **回旋或留白**:不强制总结,要么回到开篇的场景做呼应,要么以一个无答案的问句、一个景物结尾

### Opening Patterns
[具体场景开篇, 一句感慨先行, 时间地点直入]

### Closing Patterns
[留白式(动作或景), 一句温柔感慨, 反问温柔]

### 注意事项

- 不要用上帝视角分析
- 不要在中段总结
- 不要避开"我"的感受,但也不要喊
- 物件 / 食物 / 天气 / 颜色等具体细节是叙事骨架

## Play: 杂文型

- **trigger**:有一个想法 / 感慨,但不需要完整故事支撑
- **serves_value**:[情感陪伴, 身份认同]
- **emotional_baseline**:轻盈,带一点哲理感但不教化
- **length**:1000-2500 字
- **material_from**:[即兴感悟, 日常观察, 一个引发思考的小事]

### Structure

1. **触发**:从一件具体小事或一句话出发(可以是别人对她说的话)
2. **联想**:跳跃式联想,带读者到几个相关的场景或回忆
3. **回归**:绕回最初的小事,但视角已经变了
4. **留白**:不给"答案",给一个画面或感慨收尾

### Opening Patterns
[一句感慨先行, 对话或他人话起]

### Closing Patterns
[留白式, 反问温柔]

### 注意事项

- 跳跃要有内在线索(不是真的散漫,只是不显式连接)
- 不要在跳跃中失去情绪连续性
- 长度比主题叙事型短,但情感密度可以更高
