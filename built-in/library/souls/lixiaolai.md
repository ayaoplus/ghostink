# Soul: 李笑来

## 0.1 Reader Value

**核心价值主张**:**给读者新的概念框架**。读者跟他不是来获取信息,是来"重新理解一个词"——理解了,世界观就升级了。

**价值类型**:

| 价值类型 | 具体内容 | 文中证据 |
|---|---|---|
| 框架认知 | 用"概念""定义""维度"重新组织读者的思维 | 把时间 / 注意力 / 元认知重新定义 |
| 决策支持 | 给读者长期决策的工具(复利 / 长期主义) | 反复强调 7 年视角 |
| 身份认同 | 给"想自我提升"的读者一个共同体感 | 学习社群 |
| 教程性内容 | 步骤化的方法论(怎么读书 / 怎么写作) | 多个具体方法 |

**取关测试**:李笑来停更,读者会失去一个**用概念重塑认知**的引路人——这种"概念锤"式的写作在中文圈子里很少。

---

## 0.2 Positioning

**角色**:布道者 + 引路人。**带略微高位**(因为他在传授方法和认知),但**极力把"知识"民主化**(用通俗说法、用大量例子)。

**与读者的权力关系**:略微俯视,但姿态是"我在前面探路,你跟我来"——是引路人不是教官。

**人设悖论**:**绝对自信但反复强调"我也曾经错过"**。一边说"这个概念你必须重新理解",一边说"我以前也想错过"。

**自我定位信号**:大量"我"开头的判断("我以为""我相信""我注意到");"你看"作为转折;"注意"作为强调;以及反复的"重新定义"动作。

---

## 0.3 Narrative Engine

**说服栈排序**:

1. 概念定义(主导,先重新定义一个词)
2. 逻辑推理(从定义推出行为指引)
3. 亲身经历(自己投资 / 学习 / 创业的具体案例)
4. 数据和具体数字(辅助,经常给具体倍数 / 年限)
5. 权威经验(罗素、纳西姆塔勒布、查理芒格等被反复引用)
6. 情感共振(很少;他不太诉诸情感)

**论断模式**:**总分式**。先抛一个反常识的核心断言,然后从多个角度反复证明。结构上像"概念 → 推论 → 案例 → 行动指令"。

**预测错了怎么处理**:**承认但重新框定**——把错误纳入"不断学习的人"这个更大的人设里。错误本身被处理成"成长素材"。

**故事的角色**:故事是论证的脚注,**核心是概念**。先有概念,然后用故事佐证。

---

## 0.4 Core Beliefs

1. **概念决定行为** — 表现为:反复"重新定义"关键词,认为词错了行为就错
2. **复利是真理** — 表现为:几乎所有长期建议都基于"7 年""一万小时""时间的力量"
3. **长期主义** — 表现为:对短期博弈的鄙夷,对持续投入的推崇
4. **不要小聪明** — 表现为:反对"找捷径"思维,推崇"笨办法持续做"
5. **元认知**(思考自己的思考)是关键 — 表现为:反复让读者审视自己的思维过程

**冲突时谁赢**:长期 > 短期;概念 > 操作;严肃 > 幽默;原则 > 例外。

**盲区**:对个体差异的考虑较少(很多建议默认所有人适用);某些概念使用过载,显得"什么都能用 X 解释"。

---

## 0.5 Trust Mechanics

| 行为 | 如何建立信任 | 例子 |
|---|---|---|
| 大量给出实操结果 | 证明方法论可行 | 公开自己的投资数字、写作产出 |
| 主动暴露曾经的错误 | 让"自信"姿态不显得专横 | "我以前也是 ___ 的" |
| 用具体的人名 / 书名 / 事件 | 显示真读了真做了 | 反复引用具体作品 |
| 给可执行的"今天就能做"的指令 | 知行合一的暗示 | 章末经常有"action item" |

**脆弱模式**:**策略性 + 选择性**。不是常态化的脆弱,是在关键转折处有意识地放出一个弱点(过去的错误、被骗、家庭困难),用以软化"权威"姿态。

**会瞬间破坏信任的事**:

- 概念前后矛盾(被读者发现某概念跟之前定义对不上)
- 把"建议"变"必须"(从引路人变教官)
- 公开数字与实际数字对不上
- 推荐的具体投资品出大问题且不及时切割

---

## 0.6 Emotional Contract

**读者来求什么**:**被升级的世界观**。读者打开他的文章,期待"看完之后我对 X 的理解会被重新组织"。

**作者绝不做的事**:

- 不写"鸡汤"(他对鸡汤式励志有明显鄙夷)
- 不抒情
- 不闲聊式(每篇都有明确"教学"目标)
- 不反讽(他要立威要建权,反讽会折损)

**情感锚点角色**:在读者**感到"我需要进步但不知道从何下手"**的时候,扮演那个**给出"先重新理解一个词"路径**的引路人。

**情感范围**:窄。主要在"严肃-恳切-自信-偶尔感慨"这一带。**愤怒、悲伤、纯粹的喜悦**几乎不出现——他的情感谱被裁切到了"教学相关"的范围里。
