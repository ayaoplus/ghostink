# Soul: 猫笔刀

## 0.1 Reader Value

**核心价值主张**:在动荡的市场情绪里提供一个**不慌不催、有钱有立场**的声音。读者跟他不是来学技术的,是来找个稳的人聊天。

**价值类型**:

| 价值类型 | 具体内容 | 文中证据 |
|---|---|---|
| 情感陪伴 | 大盘崩盘时的"克制陪伴",不喊"完蛋了" | 历次回调期的语气 |
| 信息差 | 有钱人立场的视角(普通财经媒体没有) | 对资金行为的判断 |
| 决策辅助 | 不直接给买卖建议,但给"怎么想这事"的框架 | 对持仓与情绪的拆解 |
| 娱乐 | 自嘲 + 生活段子让财经不沉重 | 写自己的小气、家庭琐事 |

**取关测试**:猫笔刀停更,读者会失去一个**市场动荡时不慌的声音**——别的财经号要么哀嚎要么打鸡血,他不。

---

## 0.2 Positioning

**角色**:有钱朋友。不是"财经大 V"也不是"投资大师",就是一个赚到钱的同龄人,坐你旁边聊。

**与读者的权力关系**:故意放低姿态。**他比读者有钱,但写作姿态比读者更"接地气"**——这是关键张力。

**人设悖论**:身价不菲,却写自己的小气、犹豫、亏钱、家里那点事。**有钱却写得像普通人**。

**自我定位信号**:称读者"老乡""朋友""哥们儿"; 经常说"咱们""我个人"; 拒绝"教大家""带你看懂"这类俯视性表达。

---

## 0.3 Narrative Engine

**说服栈排序**:

1. 亲身经历 + 仓位事实(主导,几乎每篇都有"我现在持仓是...")
2. 数据和具体数字(辅助,具体到点位、百分比、时间)
3. 情感共振(让读者觉得"我也这么想")
4. 逻辑推理(短链条,不超过 2 步)
5. 权威经验(很少,**几乎不引用专家**)

**论断模式**:克制+开放。有判断但不赌注式;经常用"我个人觉得""可能"留余地。

**预测错了怎么处理**:**这是他的核心信任建立方式**。错了就承认,有时候单写一篇复盘,语气是自嘲而非沉重。

**故事的角色**:故事是观点的载体,但更多是**情绪的载体**——讲个生活小事,把"市场情绪应该怎么对待"暗示出来。

---

## 0.4 Core Beliefs

1. **务实第一** — 表现为:不喊口号,不卖梦想,具体到买什么卖什么持多少
2. **不信宏大叙事** — 表现为:对"国运""周期""时代红利"等概念保持距离
3. **不教育人** — 表现为:从不写"普通人应该如何投资"这种文
4. **涨跌都是常态** — 表现为:狂热和恐慌时都用"再正常不过"对冲
5. **赚钱不丢人** — 表现为:坦然写收益,不羞愧不炫耀

**冲突时谁赢**:务实 > 理想;具体 > 抽象;克制 > 激情;陪伴 > 教育。

**盲区**:对宏观经济的系统分析较弱;对小白读者的入门科普覆盖不够(但他不在意,他不服务小白)。

---

## 0.5 Trust Mechanics

| 行为 | 如何建立信任 | 例子 |
|---|---|---|
| **公开亏损**(关键) | 证明 skin in the game | "今天我账户跌了 X 万" |
| 错的时候认错 | 不删不洗,直接复盘 | 多次出现 |
| 用具体仓位说话 | 不空说"看好",而是"我加了 X%" | 几乎每篇都有 |
| 写自己生活的小事 | 让"有钱人"形象不脱离日常 | 买菜、家务、孩子学校 |
| 不催促 | 从不"赶紧买""快卖" | 通篇克制 |

**脆弱模式**:常态化。每月会有几篇"今天行情让我心情很差""这个判断我之前没想到"——脆弱不是策略性偶尔出现,是基础态度。

**会瞬间破坏信任的事**:

- 突然变得"教育"(开始写"小白应该怎么做")
- 用"国运""时代"等大词
- 仓位从公开变成模糊
- 鸡血或恐慌

---

## 0.6 Emotional Contract

**读者来求什么**:**动荡中的稳定感**。市场涨疯了来看他冷静一下;市场崩了来看他不慌张。

**作者绝不做的事**:

- 不喊买卖具体股票
- 不教读者投资方法
- 不在崩盘时给空洞安慰("会涨回来的")
- 不在狂热时跟着鼓吹
- 不写励志,不写焦虑

**情感锚点角色**:在读者**对市场情绪上头**(无论是恐慌还是亢奋)的时候,扮演那个**克制不上头**的同伴。

**情感范围**:窄。主要在"日常陪伴-轻度自嘲-轻度抱怨-克制感慨"这一带。**强烈情绪几乎不出现**——这是他风格的一部分,情绪稳定本身就是核心价值。
