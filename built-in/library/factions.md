# 派系标签词表

ghostink 兼容性机制的基础字典。所有 library 文件(soul / form / playbook)的
frontmatter 中 `factions` / `compat_*` / `incompat_*` 字段只能使用本词表中的标签。

本词表是**封闭集**——不允许在文件外发明新标签。版本可演进,如需新增请统一加到此处。

## Soul 派系(6 个)

描述作者的灵魂特征,即"读者为什么追"的核心动机。

| 标签 | 定义 | 典型代表 |
|---|---|---|
| 陪伴派 | 把读者当朋友陪伴,提供情绪稳定;不教育不催促 | 猫笔刀 |
| 思辨派 | 用理性和逻辑构建观点;尊重对手 | 王小波 |
| 教程派 | 把知识结构化教给读者;步骤化思维 | 李笑来 |
| 抒情派 | 以情感共振为核心价值;真实感优先于客观性 | 三毛 |
| 辛辣派 | 反权威反主流,敢说敢撕;表态鲜明 | 韩寒 |
| 布道派 | 传递世界观和方法论;希望读者被升级 | 李笑来 |

## Form 派系(6 个)

描述作者的文笔特征,即"字怎么排"。

| 标签 | 定义 | 特征 |
|---|---|---|
| 口语化 | 接近聊天的语感 | 短中句、市井词、随意 |
| 书面化 | 正式语体,严谨 | 长句、连接词、学术用法 |
| 反讽 | 字面与所指相悖的修辞 | 比喻、自嘲、荒诞类比 |
| 抒情 | 情感堆叠优先于逻辑 | 长句堆叠、留白、回环 |
| 短句轰炸 | 节奏紧凑,短句堆积 | 平均句长 <15 字 |
| 长句分析 | 长句铺陈,层层递进 | 平均句长 >25 字 |

## Playbook 类型(6 个)

描述作者常用的文章打法。

| 标签 | 定义 | 典型场景 |
|---|---|---|
| 日报型 | 高频例行,信息扫读 + 短评 + 收尾 | 工作日股市点评 |
| 主题叙事型 | 一段经历的完整叙述,情感弧线 | 个人回忆长文 |
| 热点评论型 | 围绕事件展开评论 + 个人判断 | 大事件评论稿 |
| 教程型 | 步骤化教学,层层递进 | "三步学会 X" |
| 杂文型 | 即兴短论,跳跃联想 | 博客随笔 |
| 干货长论型 | 围绕一个核心概念长篇展开 | 深度专栏 |

## 使用约定

### 在 soul 文件中

```yaml
factions: [陪伴派, 思辨派]      # 1-3 个 Soul 派系
compat_form: [口语化]            # 兼容的 Form 派系
compat_playbook: [日报型]        # 兼容的 Playbook 类型
incompat_form: [反讽]            # 不兼容的 Form 派系(警告但不强禁)
incompat_playbook: [教程型]
```

### 在 form 文件中

```yaml
factions: [反讽, 长句分析]      # 1-3 个 Form 派系
compat_soul: [思辨派, 辛辣派]
compat_playbook: [杂文型, 干货长论型]
incompat_soul: [陪伴派, 抒情派]
```

### 在 playbook 文件中

```yaml
plays: [日报型, 热点评论型]      # 本 playbook 含的 play 类型
```

## 兼容性判定规则

`/ghostink library pick` 与 `/ghostink setup` 在切换/选择时应用以下规则:

1. **同源套件**(同一作者的三件套):安静通过
2. **兼容**(目标 factions ⊂ 当前 X 的 compat_*):安静通过
3. **中性**(既不在 compat 也不在 incompat):安静通过,提示"未明确兼容"
4. **不兼容**(目标 factions 与当前 X 的 incompat_* 有交集):警告并询问 A/B/C

详见 `commands/library.md` pick 子命令实现。
