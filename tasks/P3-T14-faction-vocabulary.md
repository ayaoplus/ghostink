ghostink 兼容性机制的基础字典。所有 library 文件的 frontmatter

## Soul 派系

| 标签 | 定义 | 典型代表 |
|---|---|---|
| 陪伴派 | 把读者当朋友陪伴,提供情绪稳定 | 猫笔刀 |
| 思辨派 | 用理性和逻辑构建观点 | 王小波 |
| 教程派 | 把知识结构化教给读者 | 李笑来 |
| 抒情派 | 以情感共振为核心价值 | 三毛 |
| 辛辣派 | 反权威反主流,敢说敢撕 | 韩寒 |
| 布道派 | 传递世界观和方法论 | 李笑来 |

## Form 派系

| 标签 | 定义 | 特征 |
|---|---|---|
| 口语化 | 接近聊天的语感 | 短中句、市井词、随意 |
| 书面化 | 正式语体,严谨 | 长句、连接词、学术用法 |
| 反讽 | 字面与所指相悖的修辞 | 比喻、自嘲、荒诞类比 |
| 抒情 | 情感堆叠优先于逻辑 | 长句堆叠、留白、回环 |
| 短句轰炸 | 节奏紧凑,短句堆积 | 平均句长 <15 字 |
| 长句分析 | 长句铺陈,层层递进 | 平均句长 >25 字 |

## Playbook 类型

| 标签 | 定义 | 典型场景 |
|---|---|---|
| 日报型 | 高频例行,信息扫读 | 工作日股市点评 |
| 主题叙事型 | 一段经历的完整叙述 | 个人回忆长文 |
| 热点评论型 | 围绕事件展开评论 | 大事件评论稿 |
| 教程型 | 步骤化教学 | "三步学会 X" |
| 杂文型 | 即兴短论,跳跃联想 | 博客随笔 |
| 干货长论型 | 围绕概念长篇展开 | 深度专栏 |
